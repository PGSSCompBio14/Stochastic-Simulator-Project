Stochastic Simulator Project

Biological Operations modeled through Stochastic Simulation

Download as .zip Download as .tar.gz View on GitHub

Welcome to BOSS.

BOSS's funtion is to model a biological system through stochastic simulations. All you need to get started is an input file and our easy to use code on Github. Either download as a .zip file with the top left button or press the button in the top right corner to get started. If you choose to go to Github, there will be a sample input file under the name README to provide the correct format we require. If you choose a .zip file, there will be a README file saved with the simulator code.
