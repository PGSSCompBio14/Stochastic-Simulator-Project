Welcome to BOSS.

Description

BOSS is a tool for modeling stochastic reactions. Over the course of five weeks, our PGSS team created BOSS using Python. We implemented the Gillespie Algorithm to realistically model the relationships of various natural species with each other. By taking into account the factor of randomness and natural fluctuations, our model proves to be more accurate than a model that merely uses Ordinary Differential Equations (ODEs).

How It Works

The user inputs the necessary information, including initial molecule counts, number of iterations, and reactions. BOSS uses this data to update the molecule counts over the indicated duration, viewing time through individual events, instead of as a continuous line. The next reaction to be executed and the time interval between reactions are chosen using random numbers, which helps make the results more realistic. Applications of BOSS include, but are not limited to, the Lotka-Volterra predator-prey model, the effect of competitive inhibitors on HIV Protease-1, and diffusion across a cell membrane.
